FAQ

Tens dúvidas?
Nós temos respostas.

Tudo o que precisas de saber antes de pedires acesso antecipado.

O que é o acesso antecipado? ▾
O acesso antecipado é um programa limitado para os primeiros utilizadores que querem experimentar o BoltBoost antes do lançamento oficial. Vais ter acesso à plataforma completa, dar feedback e ajudar a moldar o produto. As vagas são limitadas e a seleção é feita pela nossa equipa.
Quanto custa o BoltBoost? ▾
Durante o período de acesso antecipado, o BoltBoost é completamente gratuito para todos os utilizadores selecionados. Após o lançamento oficial, haverá um plano gratuito com funcionalidades base e planos premium para funcionalidades avançadas. Os utilizadores early access terão condições especiais.
Como funciona a inteligência artificial? ▾
A IA do BoltBoost analisa as tuas notas, identifica padrões e sugere como organizar o teu dia. Pode priorizar tarefas automaticamente, agrupar notas relacionadas e até sugerir o melhor momento para fazer follow-up com clientes. Tudo de forma privada — os teus dados nunca são usados para treinar modelos externos.
Os meus dados estão seguros? ▾
Sim, a 100%. Utilizamos Row Level Security (RLS) no Supabase, o que significa que tecnicamente é impossível aceder às notas de outro utilizador. Os dados são encriptados em trânsito e em repouso. Não vendemos nem partilhamos os teus dados com terceiros, nunca.
Funciona no iPhone e Android? ▾
Sim. O BoltBoost é uma Progressive Web App (PWA) — funciona em qualquer browser, em qualquer dispositivo. No iPhone podes adicioná-lo ao ecrã principal através do Safari (Partilhar → Adicionar ao ecrã inicial). No Android o processo é similar. A experiência é idêntica a uma app nativa.
Preciso de instalar alguma coisa? ▾
Não. O BoltBoost funciona diretamente no browser — Chrome, Safari, Firefox, Edge. Não precisas de instalar nada. Se quiseres uma experiência mais próxima de uma app, podes adicioná-lo ao ecrã principal do teu telemóvel com um toque.
Posso usar em equipa? ▾
Neste momento, o BoltBoost é uma solução individual. Cada conta é privada e os dados não são partilhados entre utilizadores. A funcionalidade de equipas e colaboração está no nosso roadmap e será lançada numa fase posterior. Os utilizadores early access serão os primeiros a ter acesso.
Como funcionam os lembretes? ▾
Podes definir um lembrete para qualquer nota, com data e hora específicas. O BoltBoost verifica automaticamente os teus lembretes e envia uma notificação diretamente no browser (e no telemóvel se tiveres a app no ecrã principal). Precisas de dar permissão para notificações na primeira vez que usas.
Posso exportar as minhas notas? ▾
A funcionalidade de exportação está em desenvolvimento e será incluída numa das próximas atualizações. Suportaremos exportação em PDF, Markdown e JSON. Os teus dados são sempre teus e podes pedir uma exportação completa em qualquer momento.
Quando vou receber acesso após preencher o formulário? ▾
A nossa equipa analisa os pedidos semanalmente. Entraremos em contacto por email ou telefone no prazo de 3 a 5 dias úteis. Se fores selecionado, receberás instruções de acesso imediatamente. As vagas são limitadas, por isso não esperes — pede já o teu acesso.
